# HUNT
HUNT Unreal Engine Projet for Bros only

1) Install and update GIT
"git update-git-for-windows"

2) Clone repo
"git clone https://github.com/lhocquemiller/HUNT.git"

3) Install Epic Game laucher

4) From Epic Game Launcher, install Unreal Engine 4.26

5) You can now open the project with Unreal Engine
