# HUNT
HUNT Unreal Engine Project for Bros

TOOLS:
1) Mindmeister
https://mm.tt/1818554815?t=KDHwkMZ2Ik

INSTALLATION:
1) Install and update GIT
"git update-git-for-windows"
2) Install GIT Lfs: https://git-lfs.github.com/
3) Run from console "git lfs install"
4) Clone repo
"git clone https://github.com/lhocquemiller/HUNT.git"
3) Install Epic Game laucher
4) From Epic Game Launcher, install Unreal Engine 4.26. You may want to install it on SSD.
5) You can now open the project with Unreal Engine

GUIDELINES:
1) Do not add features directly on master, create new branches.
2) Always build and test before merging.
3) Try to fix warnings before merging...

PUSHING:
What to do if I have a large file (>100MB)?
-> Track it with GIT Lfs:
"git lfs track "filepath/filename"
